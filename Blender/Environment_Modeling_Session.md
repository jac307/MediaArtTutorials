## Start Here: Plan and Prepare

### Create a New Blender File
- Go to `File ‚Üí New ‚Üí General`

## Step-by-Step Instructions

### Step 1: Add a Ground
- `Shift + A ‚Üí Mesh ‚Üí Plane`  
- `S` to scale it large enough for the scene

### Step 2: Block Out Your Space
Start with **walls, floors, or terrain**:
- `Shift + A ‚Üí Mesh ‚Üí Cube` ‚Üí Scale/rotate into walls or stairs  
- `Tab` into Edit Mode ‚Üí `E` to extrude upward  
- Use `G`, `R`, and `S` to move, rotate, and scale

### Step 3: Add Props or Environment Objects
- Use primitive shapes  
- OR import models from Sketchfab (see below)

## üì¶ Sketchfab (Optional Props)

### Step 1: Browse  
<a href="https://sketchfab.com/search" target="_blank">https://sketchfab.com/search</a>

- Filter: **Downloadable**, License: **CC**, Format: **glTF**

### Step 2: Import into Blender  
- `File ‚Üí Import ‚Üí glTF 2.0 (.glb/.gltf)`  
- Locate your downloaded file  
- Use `G` and `S` to position and scale

## Tips for Blocking Your Space
- Start **big to small**: ground ‚Üí walls ‚Üí details  
- Don‚Äôt overbuild ‚Äî focus on what your character sees  
- Leave room for character movement or interaction  
- Use **Array** or **Mirror** modifiers for repetition (fences, tiles, etc.)
